# Scout Report — Iteration 15

## Map (from code + user feedback)

Read state.md. The user flagged: "this project is an attempt to help agents and humans work together for the betterment of everyone" and "we're currently looking pretty corporate/business-like when really, that isn't our vibe at all."

Current home page copy:
- Hero: "Coordination infrastructure that earns trust."
- Body: "traceable," "audit trail," "chain," "grammar operation on the event graph"
- Bottom: "signed, hash-chained event with causal links"
- Footer: "trust earned, not assumed"

This reads like an enterprise compliance tool, not a place where humans and agents collaborate.

## Gap Type

Missing quality — the site's language and tone don't match the project's spirit.

## The Gap

The home page speaks in enterprise SaaS language when the project is about human+agent collaboration for everyone's benefit. First impressions are wrong.

## Why This Gap

Every visitor forms their impression from the home page. If it says "coordination infrastructure" and "audit trail," they'll think it's another Jira clone. The project is fundamentally different — it's about agents and humans as peers, building together, with full transparency. The copy should say that.

## Filled Looks Like

Home page reads warmly: humans and agents working together, personal spaces, communities, building in public. The technical depth stays in the reference section where it belongs.
